# gtf
R data package that provides gtf files for commonly studied model organisms.  See `DESCRIPTION` file for details.  `gtf` is currently used by at least one existing R package: `geneXtendeR`. 

## Installation instructions

You can install the current GitHub version using the [devtools](https://github.com/hadley/devtools) package and the following command in R:
```R
devtools::install_github("Bohdan-Khomtchouk/gtf")
```
